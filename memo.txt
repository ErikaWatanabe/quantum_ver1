最終的に何をしたいのか : 指数と作成したポートフォリオのグラフ作成
    →　指数はS&Pとかから一定の期間指定して抽出
    →　ポートフォリオを頑張って作る！

【　各指数の特徴　】
日経平均 : 単純に株価の平均
TOPIX : 時価総額比率で加重平均
S&P 500 : 時価総額比率で加重平均


【　ポートフォリオ作成　】
例えば...
・TOPIXは2146銘柄、そこからカーディナリティ制約C = 100でまずは作ってみる
・期間は2024年4月~5月

★ ゴール
    →　指数と基準価格のグラフを作成し、比較
    →　トラッキングエラーの数値も計算して出す

★ 銘柄組み込む前に決めること
    →　カーディナリティ制約C
    →　純資産総額
    →　総口数

★ アニーリングで決めること
    →　C個の銘柄を何にするか
    →　各銘柄の購入株数

★ 制約条件で加えること
    →　市場の流動性
    →　純資産総額を十分に使えているか

★ 必要なデータ
    →　TOPIXの構成銘柄の…
        →　銘柄名
        →　構成割合
        →　最低購入金額（最小株単位が100 or 1）
        →　（分野）

★ グラフ書くには
    →　一日ずつの基準価格を算出する
    →　基準価格は各銘柄の評価利益（購入時の価格 - その日の価格）の合計を口数で割る


【　わからないこと　】
・組み入れ銘柄の決定は、追従する指数と同じタイミング(TOPIXなら毎年4月)
    →　トラッキングエラーを最小化するようなって言っても4月のちょっとの期間だけじゃ測れない
    →　トラッキングエラー小さくするように、頻繁に差を見て銘柄更新ならわかる
    →　先行研究ではトラッキングエラーを最小化するような式を組んでたけど、できる？
・4月の時点で何を最小化するようなポートフォリオを作ればいいの？？？？？？？？
・基準価格を計算するために、架空のインデックスファンドの純資産総額と総口数はどう決める？
・1ヶ月ごとにトラッキングエラー見て、銘柄少し入れ替えるとか？
・予算を十分に使うために、純資産内で購入額を最大化するとか？
・去年とかここ数年のトラッキングエラー見て決めるとか？（推定トラッキングエラー！！！！！）

【　できそうなこと　】
・4月に決定されたTOPIXを構成する銘柄を見てウェイトが大きいのをC=100とかで適当に組む
・適当に純資産総額と総口数を決めて半年とか短期間のグラフ書く
